# image-clusterfuck
A Python GUI app for visualizing color distributions across various color spaces for image clustering. Supports cropping and thresholding.

## Supported color spaces
* BGR
* RGB
* YUV
* YCrCb
* LAB
* LUV
* HLS
* HSV
* XYZ
